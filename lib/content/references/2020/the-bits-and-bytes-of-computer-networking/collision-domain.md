# Collision domain
-   한 순간에 하나의 기기만 통신할 수 있는 네트워크 부분
-   여러 시스템이 동시에 데이터를 전송하려고 하면 전기적인 신호가 서로 간섭될 수 있다.

- 최근의 유선 인터넷의 경우 [hub-and-switch](hub-and-switch.md) 를 통해 이를 해결하지만, 더 나은 속도를 위해서는 full-duplex link 를 필요로 한다.
- 무선 네트워크의 경우 [Carrier-sense multiple access with collision avoidance - Wikipedia](https://en.wikipedia.org/wiki/Carrier-sense_multiple_access_with_collision_avoidance) 를 통해 이를 해결한다.
