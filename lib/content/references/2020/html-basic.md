## 줄바꿈  
{{< codepen id="mqyZYQ" >}}
- HTML 에서는 본문의 내용을 그냥 줄바꿈 한다고 해서 줄바꿈이 생기지 않는다. 
- `<br>` 태그를 이용해 줄바꿈을 할 수 있다.
	- `<br>` 태그는 앞서 배운 태그들과 달리 닫는 태그가 필요없다. 이런 태그로 `<img>, <input>, <br>, <hr>, <meta> ` 와 같은 태그들이 있다. 

- `<p> paragrapgh </p>` 단락을 바꾸는 태그 
	- 단락을 표현하고자 할때는 `<br>` 보다 이편이 더 좋은 표현이다. 
	- 대신 고정된 여백이 생기고 이에 대한 디테일을 수정할 수는 없다. 
	- 이를 극복하기 위해 css 를 사용할 수 있다.

# 부모 자식과 목록

```html
<p>
    <a href="https://opentutorials.org">opentutorials</a>
</p>
```

- 위와 같은 경우에 p 태그가 a 태그의 자식태그라고 한다.
- 꼭 이렇게 `<p>` 태그와 `<a>` 태그가 반드시 이렇게 구성되어야 하는 것은 아니다. 그런데 몇몇 태그의 경우 이렇게 고정된 관계를 가지는 태그가 있다.
  
{{< codepen id="MWgevpK" >}}
- `<li>` : 리스트를 나열할 때 쓰이는 태그 
- `<ul>` : 리스트와 리스트를 구분하기 위한 경계 태그 
- `<li>` 태그와 `<ul>` 태그는 서로 의존적이다. 부모자식의 관계로 묶어줄 필요가 있다.

- `<ol>` : 순서가 있는 리스트를 나열할때 쓰이는 태그

## 문서의 구조와 슈퍼스타
```html
<title>WEB1 - html</title>
<meta charset="utf-8">

<ol>
  <li>HTML</li>
  <li>CSS</li>
  <li>JavaScript</li>
</ol>
```
- `<title></title>` 태그는 검색엔진이 주로 참고하기 때문에 안쓰면 손해! 
- `<meta charset = "utf-8">` : 브라우저에게 인코딩을 알리는 방법

{{< codepen id="gXpoVa" >}}

- 본문은 `<body>` , 본문을 설명하는 태그는 `<head>` 
- 그리고 이 둘을 `<html>` 태그로 감싼다. 
- `<!doctype html>` 을 통해 현재 문서가 html 이라는 것을 알린다. 
- 이 태그들이 가장 많이 사용되는 태그들이다.
