# HandlerInterceptors vs. Filters in Spring MVC
![](https://www.baeldung.com/wp-content/uploads/2021/05/filters_vs_interceptors-768x287.jpg)  

|             | Filter                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                            | [handler-Interceptor](handler-Interceptor.md)                                                                                                                                                                                                                                                                                                                                                                           |
|-------------|-------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|-----------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------------|
| base class  | javax.servlet.Filter                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                              | org.springframework.web.servlet.HandlerInterceptor                                                                                                                                                                                                                                                                                                                                            |
| methods     | - doFilter()                                                                                                                                                                                                                                                                                                                                                                                                                                                                                   | - preHandle()<br>- postHandle()<br>- afterCompletion()                                                                                                                                                                                                                                                                                                                                        |
| execution   | - Intercept requests before they reach the DispatcherServlet                                                                                                                                                                                                                                                                                                                                                                                                                                                                                                      | - Intercepts requests between the DispatcherServlet and Controllers                                                                                                                                                                                                                                                                                                                           |
| ideal tasks | - Authentication<br>- Logging and auditing<br>- Image and data compression<br>- Any functionality we want to be decoupled from Spring MVC                                                                                                                                                                                                                                                                                                                                                                                                                         | - Handling cross-cutting concerns such as application logging<br>- Detailed authorization checks<br>- Manipulating the Spring context or model                                                                                                                                                                                                                                                |
| example     | - [FormContentFilter (Spring Framework 5.3.14 API)](https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/web/filter/FormContentFilter.html)<br>- [RelativeRedirectFilter (Spring Framework 5.3.14 API)](https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/web/filter/RelativeRedirectFilter.html)<br>- [HttpPutFormContentFilter (Spring Framework 5.3.14 API)](https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/web/filter/HttpPutFormContentFilter.html) | - [ThemeChangeInterceptor (Spring Framework 5.3.14 API)](https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/web/servlet/theme/ThemeChangeInterceptor.html)<br>- [LocaleChangeInterceptor (Spring Framework 5.3.14 API)](https://docs.spring.io/spring-framework/docs/current/javadoc-api/org/springframework/web/servlet/i18n/LocaleChangeInterceptor.html) |
- [delegating-filter-proxy](delegating-filter-proxy.md) is also Filter
- Any functionalities that can be implemented without spring mvc may be implemented as Filter
