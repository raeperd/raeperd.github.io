# 10 Difference between Primitive and Reference variable in Java
|                       | primitive       | reference              |
|-----------------------|-----------------|------------------------|
| Default value         | true, 0, 0.0    | null                   |
| Null                  | never           | possible               |
| What do they store    | values          | handles (not pointer)  |
| Assign value using =  | copy value      | copy handle            |
| ==                    | compare value   | compare address        |
| Pass as arguments     | passed by value | pass by value (handle) |
| Stack vs Heap         | stack           | heap                   |
| Memory consumption    | less            | more (object headers)  |k
